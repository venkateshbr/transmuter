Value Creation Platform: Comprehensive PRD & Feature List

1. Product Overview
Product Name: Transmuter Tagline: Value Creation Platform for Transformation Programs Positioning: AI-powered command center for enterprise transformation programs Operated By: ishirock Logged-in User Context: Transformation Office (role-based access)
Core Purpose: Transmuter enables transformation offices to manage, track, and govern enterprise initiatives end-to-end — from ideation through scoping, in-progress execution, and completion — with real-time visibility into value creation, milestones, risks, KPIs, financials, and meeting cadences.

3. Module 1 — My Dashboard
URL: / Purpose: Personalized at-a-glance overview of the user's initiatives, milestones, and actions
3.1 Filters
Three dropdown filters that update all dashboard widgets simultaneously:
* All Business Units — filter by organizational BU
* All Workstreams — filter by workstream
* All RAG — filter by Red/Amber/Green status
3.2 Executive Summary Button
* Orange primary CTA button: + Executive Summary
* Generates/exports a portfolio-level executive summary document
3.3 My Milestones Widget
* Shows milestones assigned to the logged-in user across all initiatives
* Badge shows total count (e.g., "2")
* Each milestone row shows: milestone name, parent initiative name, due status ("Due today" in red, or relative days)
* Clicking a milestone navigates to the parent initiative detail
3.4 My Actions Widget
* Shows action items assigned to the logged-in user from meeting sessions
* Badge shows total count (e.g., "3")
* Each action shows: action description, source meeting + date, due status
* Clicking an action navigates to the source meeting
3.5 Pipeline Value Card
    * Displays portfolio-level financial summary:
    * Plan value (e.g., $0m)
    * Actual value
* Links to Initiatives Pipeline
3.6 Pipeline by Stage Widget
* Visual breakdown of initiatives by lifecycle stage (Scoping → In-Progress → Complete)
* "View Pipeline" link → navigates to full pipeline
* Tooltip: "What is Pipeline by Stage?"
3.7 Summary Stat Cards
* Total Initiatives — total count, links to pipeline
* At Risk — count of red-RAG initiatives, links to pipeline filtered by red
* Pending Approvals — count of initiatives awaiting gate approval, links to governance
3.8 RAG Breakdown Widget
* Visual breakdown of all initiatives by Red/Amber/Green health status
* Tooltip explanation available
3.9 Portfolio Pressure Gauge
* Semi-circular gauge visualization (0–10 scale)
* Displays aggregate portfolio pressure score
* Shows "Low/Medium/High" classification
* Shows average of active initiatives
* Tooltip: "What is Portfolio Pressure?"
3.10 Risk Heatmap Widget
* 3×3 matrix grid: Impact (High/Medium/Low) vs Likelihood (Low/Medium/High)
* Each cell shows count of risks at that intersection
* Cells are color-coded: red=High, orange=Medium, teal/green=Low
* Each cell is clickable → links to /pmo/risks filtered by that impact/likelihood combination
* "Open register →" link to full risk register
* Tooltip: "What is Risk Heatmap?"
3.11 KPI Pulse Widget
* Shows KPIs with actuals across all initiatives
* Currently shows placeholder if no actuals have been entered
3.12 Value Bridge Widget
* Three-column layout: Benefits, Costs, Net
* Each column shows: Base Case, High Case, Actual rows (in USD)
* Legend: Base · High · Actual
* Tooltip: "What is Value Bridge?"
3.13 Recent Activity Widget
* Shows status update activity across all the user's initiatives
* Shows count of updates
* Tooltip: "What is Recent activity?"

14. Notable UX Patterns
* Inline Expansion: Milestones and risks expand in-place rather than navigating away
* Pressure Scoring: Every milestone and initiative has a numerically computed "pressure" score with transparent sub-component breakdowns and tooltips explaining methodology
* Dual Financial Cases: All financial entries support Base Case + High Case + Actual for scenario planning
* RAG System: Red/Amber/Green health classification pervasive across all entities
* Context Help: Every dashboard widget has a "?" button with explanatory tooltip
* AI Badges on Meetings: Sessions display "TRANSCRIPT" and "AI OPTIMISED" badges to indicate enriched content
* Nudging System: Platform can automatically detect and send nudges to initiative owners who haven't submitted status updates, with a full Nudge Log
* "Nuclear" Status: Beyond "Overdue," the platform uses "Nuclear" as a severity escalation for severely late status updates
